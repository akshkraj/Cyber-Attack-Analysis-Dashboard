# 🛡️ Cyber Attack Dashboard

An interactive dashboard designed to analyze and visualize cyber attack data. This project provides insights into attack patterns, targeted industries, affected countries, and threat severity levels through data analytics and visualization.

## 📊 Features

* Cyber attack trend analysis
* Attack type distribution
* Country-wise attack analysis
* Industry-wise target analysis
* Severity level monitoring
* Interactive dashboard visualizations
* Actionable cybersecurity insights

## 🛠️ Tech Stack

* Python
* Pandas
* NumPy
* Power BI
* Jupyter Notebook

## 📂 Dataset

The dataset includes information such as:

* Attack Type
* Source Country
* Target Country
* Industry Sector
* Date and Time
* Severity Level
* Attack Status

## 📈 Dashboard Insights

* Total Number of Cyber Attacks
* Attack Type Distribution
* Top Targeted Countries
* Top Targeted Industries
* Monthly Attack Trends
* Severity Level Breakdown
* Industry Risk Analysis

## 🚀 Project Workflow

1. Data Collection
2. Data Cleaning and Preprocessing
3. Exploratory Data Analysis (EDA)
4. Data Visualization
5. Dashboard Development
6. Insight Generation

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/akshkraj/Cyber-Attack-Dashboard.git
```

Navigate to the project folder:

```bash
cd Cyber-Attack-Dashboard
```

Install dependencies:

```bash
pip install -r requirements.txt
```
